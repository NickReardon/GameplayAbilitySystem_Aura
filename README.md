# Unreal Engine 5.6 Gameplay Ability System - Top-Down RPG 
[![Unreal Engine](https://img.shields.io/badge/Unreal%20Engine-5.6-blue)](https://www.unrealengine.com/)
[![C++](https://img.shields.io/badge/C%2B%2B-17-00599C.svg)](https://isocpp.org/)
[![Rider](https://img.shields.io/badge/IDE-JetBrains%20Rider-000000.svg)](https://www.jetbrains.com/rider/)

This repository documents my journey learning Unreal Engine 5.6 and the Gameplay Ability System (GAS) through hands-on development of a top-down RPG. Following the comprehensive [Udemy course by Stephen Ulibarri](https://www.udemy.com/course/unreal-engine-5-gas-top-down-rpg/), this project showcases my progression from UE5 fundamentals to advanced GAS implementation.

---

## Learning Objectives

- **Unreal Engine 5.6** fundamentals and advanced features
- **Gameplay Ability System (GAS)** architecture and implementation
- **C++ Integration** with Blueprint systems
- **Multiplayer** client and server replication
- **Top-Down RPG** gameplay mechanics and design patterns
- **Modern Game Development** workflows and best practices
